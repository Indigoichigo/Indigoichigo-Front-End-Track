（一）說明
1.這裡的檔是template不要動!
這個template可以處理以下檔案
html,css,scss,各種img,babel,es6,react


（二）使用步驟
1.下載node.js
https://nodejs.org/en/

2.cmd中下 $ npm -v
若低於4.2.x版更新到4.2以上

3.
將以下檔案複製到專案資料夾

src資料夾
package.json
package-lock.json
webpack.config.js

4.cmd中下 $ npm i -D -l
i=install
-D=--save-dev
-l=??(全部安裝)

5.cmd中下
5.1 $ npm run dev
變更檔案儲存時會即時更新

5.2 $ npm run build
壓縮檔案到dist

平時工作區域在src
